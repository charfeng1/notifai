## 核心价值

### 解决的问题
现代手机用户每天收到大量通知，重要信息淹没在促销、系统提示中。现有方案依赖云端处理，存在隐私泄露风险，且需要网络连接。NotifAI 使用端侧微调大模型（Qwen3-0.6B）实现零延迟、完全离线的智能通知分类，让用户专注于真正重要的消息。

### 目标用户
- 信息过载的手机用户
- 注重隐私的用户
- 需要高效通知管理的数字游民/远程工作者
- Android 开发者和 AI 工程师（作为端侧 LLM 部署参考）

### 独特优势
1. **完全端侧推理** - 无需网络，零隐私风险
2. **微调小模型** - 0.6B 参数实现 94% 分类准确率
3. **极致性能优化** - 多架构编译 + KV 缓存，15+ tok/s 推理速度
4. **涌现泛化能力** - 支持用户自定义文件夹，无需重新训练
